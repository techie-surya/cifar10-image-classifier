# 🚀 CIFAR-10 Image Classifier using PyTorch

## 📌 Overview

This project implements a Convolutional Neural Network (CNN) to classify images from the CIFAR-10 dataset.

## 📊 Dataset

CIFAR-10 contains 60,000 images across 10 classes:
Airplane, Car, Bird, Cat, Deer, Dog, Frog, Horse, Ship, Truck.

## 🧠 Model Architecture

* 2 Convolutional Layers
* ReLU Activation
* Max Pooling
* Fully Connected Layers

## ⚙️ Training Details

* Optimizer: Adam
* Loss Function: CrossEntropyLoss
* Epochs: 10

## 📈 Results

* Final Accuracy: **72.34%**
* Loss decreased consistently during training

## 🔍 Key Observations

* CNN effectively captures image features
* Pooling reduces dimensionality
* Model generalizes well on test data
